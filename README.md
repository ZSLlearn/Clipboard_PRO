# ClipboardPRO

一款专为 Windows 设计的轻量级剪贴板管理工具，自动记录您的剪贴板历史，让复制粘贴更高效。

## 核心功能

### 📋 剪贴板管理
- **自动监控** - 实时记录剪贴板变化（文本/图片/文件路径）
- **智能去重** - 自动合并重复内容，更新时间戳
- **持久化存储** - 历史记录自动保存，重启不丢失

### 🔍 高效操作
- **复选框多选** - 批量选择，一键复制或删除
- **智能搜索** - 实时搜索，快速定位历史记录
- **类型筛选** - 按全部/文本/文件/图片快速筛选
- **长文本折叠** - 自动折叠长文本，点击展开查看详情

### ⚙️ 灵活配置
- **保留条数** - 10/20/30/40/不限制（默认不限制）
- **存留时间** - 5-60 分钟可选（默认不限制）
- **退出清理** - 可选退出时清除所有数据（默认开启）

### 💾 智能清理
- 自动删除过期记录及对应图片文件
- 启动时清理孤立图片文件
- 支持右键菜单快速操作

## 快速上手

1. 下载 `ClipboardPRO.exe`
2. 双击运行（无需安装 Python）
3. 开始自动监控剪贴板

## 快捷键

| 快捷键 | 功能 |
|--------|------|
| **Ctrl+Alt+V** | 显示/隐藏主窗口 |
| **Enter** | 复制选中记录 |
| **Delete** | 删除选中记录 |
| **Ctrl+A** | 全选记录 |
| **Ctrl+F** | 聚焦搜索框 |
| **Esc** | 清空搜索 |

## 系统要求

- **系统**: Windows 10 或更高版本（推荐 Windows 11）
- **内存**: 建议 2GB 以上
- **空间**: 约 100 MB

## 技术栈

- Python 3.12
- PyQt6
- Pillow
- pywin32
