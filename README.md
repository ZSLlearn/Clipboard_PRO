# ClipboardPRO

一款专为 Windows 设计的剪贴板管理软件，帮助您高效管理剪贴板历史记录。

## 主要功能

### 剪贴板管理
- **自动监控剪贴板** - 实时监控剪贴板变化，自动记录复制的内容
- **多类型支持** - 支持文本、图片、文件路径等多种内容类型
- **智能去重** - 使用哈希算法避免重复记录相同内容
- **历史记录持久化** - 自动保存历史记录，下次启动自动加载

### 界面功能
- **复选框多选** - 使用复选框选择多个记录，操作更直观
- **智能搜索** - 实时搜索功能，快速查找历史记录
- **类型筛选** - 按内容类型筛选（全部/文本/文件/图片）
- **长文本折叠** - 自动折叠长文本，支持展开/收起
- **图片预览** - 直接在界面中预览图片内容
- **右键菜单** - 快速复制或删除单个记录
- **系统托盘集成** - 最小化到系统托盘，方便使用

### 快捷操作
- **一键复制** - 选中记录后点击"复制到剪贴板"按钮
- **批量删除** - 支持多选批量删除记录
- **全选功能** - 一键选中所有记录

### 设置选项
- **剪贴板保留条数** - 可选择 10/20/30/40/不限制（默认不限制）
- **剪贴板存留时间** - 可选择 5/10/20/30/40/60分钟/不限制（默认不限制）
- **退出时清除数据** - 可选择退出软件时是否清除所有历史记录（默认开启）

### 数据管理
- **自动清理过期记录** - 根据存留时间自动删除过期记录
- **自动清理孤立文件** - 启动时自动清理不在记录中的图片文件
- **智能文件管理** - 删除记录时自动删除对应的图片文件
- **数据持久化** - 所有设置和历史记录自动保存

## 系统要求

- Windows 10 或更高版本（推荐 Windows 11）
- 内存：建议 2GB 以上
- 磁盘空间：约 100 MB 可用空间

## 快速开始

1. 下载 `ClipboardPRO.exe`
2. 双击运行（无需安装 Python）
3. 软件自动开始监控剪贴板

## 使用说明

### 基本操作
- 复制内容后自动记录到历史列表
- 点击复选框选择要操作的记录
- 点击"复制到剪贴板"将选中内容复制到剪贴板
- 点击"删除选中"删除选中的记录
- 点击"清空历史"清空所有记录

### 设置说明
- 点击左侧"设置"按钮打开设置对话框
- 选择剪贴板保留条数（默认不限制）
- 选择剪贴板存留时间（默认不限制）
- 选择退出时是否清除数据（默认开启）
- 点击"确定"保存设置

### 快捷键
- **Ctrl+Alt+V** - 显示/隐藏主窗口
- **Esc** - 隐藏窗口到系统托盘

### 系统托盘
- 点击托盘图标显示/隐藏窗口
- 右键托盘图标显示菜单
- 选择"退出"彻底关闭软件
