# ClipboardPRO

一款专为 Windows 11 设计的剪贴板管理软件，帮助您高效管理剪贴板历史记录。

## 主要功能

- **自动监控剪贴板** - 实时监控剪贴板变化，自动记录复制的内容
- **多类型支持** - 支持文本、图片、文件路径等多种内容类型
- **智能搜索** - 实时搜索功能，快速查找历史记录
- **类型筛选** - 按内容类型筛选（全部/文本/文件/图片）
- **历史记录管理** - 最多保存50条记录，按时间倒序显示
- **智能去重** - 使用哈希算法避免重复记录相同内容
- **长文本折叠** - 自动折叠长文本，支持展开/收起
- **系统托盘集成** - 最小化到系统托盘，方便使用
- **单实例运行** - 防止重复启动，确保程序稳定运行
- **设置管理** - 可配置最大记录数、记录保留时间等

## 系统要求

- Windows 10 或更高版本（推荐 Windows 11）
- 内存：建议 2GB 以上
- 磁盘空间：约 100 MB 可用空间

## 快速开始

1. 下载 `ClipboardPRO.exe`
2. 双击运行（无需安装）
3. 软件自动开始监控剪贴板

## 技术栈

- Python 3.12
- PyQt6
- PyInstaller
