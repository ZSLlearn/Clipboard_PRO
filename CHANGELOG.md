# 更新日志

## [1.1.0] - 2026-02-21

### 新增功能

#### 配置管理
- 新增配置管理模块，支持用户自定义设置
- 配置文件自动保存和加载
- 支持的配置项：
  - 最大记录条数（10/20/30/40条或不限制）
  - 记录保留时间（1/3/7/30天或不限制）
  - 退出时是否清除数据

#### 设置对话框
- 新增设置界面，方便用户配置软件
- 直观的选项卡式界面
- 实时保存设置更改
- 支持热键快捷打开设置（Ctrl+,）

#### 数据管理优化
- 根据配置自动清理过期记录
- 支持按记录条数限制自动清理
- 退出时可选择保留或清除历史数据
- 配置持久化存储，重启后保持设置

### 改进

- 优化了存储模块，支持配置驱动的数据清理
- 改进了主窗口的退出逻辑
- 增强了用户体验，提供更多自定义选项

### 技术更新

- 新增 `src/config.py` - 配置管理模块
- 新增 `src/settings_dialog.py` - 设置对话框模块
- 更新 `src/main_window.py` - 集成设置功能
- 更新 `src/storage.py` - 支持配置驱动的数据管理

## [1.0.0] - 2026-02-20

### 初始版本

#### 核心功能
- 自动监控剪贴板变化
- 支持文本、图片、文件路径多种内容类型
- 实时搜索和类型筛选
- 长文本折叠/展开功能
- 系统托盘集成
- 单实例运行
- 智能去重

#### 界面功能
- 主窗口显示剪贴板历史记录
- 按时间倒序排列
- 右键菜单操作（复制、删除、收藏等）
- 图片预览功能
- 文件路径显示

#### 技术栈
- Python 3.12
- PyQt6
- PyInstaller
